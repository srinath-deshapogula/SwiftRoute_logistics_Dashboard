
# 🚚 SwiftRoute Logistics Dashboard

A comprehensive **Power BI dashboard solution** designed to analyze and optimize logistics operations including **Orders, Hubs, Drivers, and Vehicles**.

---

## 📊 Project Overview

The **SwiftRoute Logistics Dashboard** provides actionable insights into logistics performance through interactive visualizations.

It helps stakeholders:
- Monitor delivery performance 📦  
- Analyze hub efficiency 🏭  
- Evaluate driver performance 🚗  
- Optimize fleet utilization 🚛

## 🎯 Key Features

### 📌 1. KPI Metrics
- Total Orders  
- On-Time Delivery Rate (%)  
- Customer Satisfaction (CSAT %)  
- Average Delivery Time (Hours)  
- Month-over-Month (MoM) comparison  

---

### 🏭 2. Hubs Analysis
- Total number of hubs  
- Orders processed vs capacity  
- Hub performance ranking  
- Processing time (daily heatmap)  

---

### 🚗 3. Drivers Analysis
- Total drivers  
- Experience vs rating (scatter analysis)  
- Drivers with most delays  
- Individual driver performance profile  
- Monthly delivery trends  

---

### 🚛 4. Vehicles Analysis
- Total vehicles  
- Active vs maintenance vehicles  
- Orders by vehicle model  
- Vehicle age vs breakdown  
- Breakdown by vehicle code & model  
- Orders by vehicle type

3. Use filters (Year, Month, Driver Name)  
4. Explore interactive visuals
